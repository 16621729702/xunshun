Lazy.js是一个将常用JS方法、AJAX方法、easyui操作方法进行总结归集的JS工具类。目的很简单，就是将前端WEB开发常用的JS操作方法归集成一个js对象，以方便今后爽快的调用。
它包括3个方法集，如果你的前端没有使用easyui，你可以在lazy.js中去除前两个方法集。
lazy.easyui.methods(); //需要引入jquery和easyui文件
lazy.ajax.methods(); //需要引入jquery和easyui文件
lazy.kit.methods(); //需要引入jquery
功能名称 前端调用方法 返回结果
自定义条件搜索，并更新结果集 lazy.easyui.search(gridId,formId) 
1、需要引入与easyui相关的文件
2、datagrid数据表格ID命名规则：id="dg-system-user-list"
3、treegrid树形表格ID命名规则：id="tg-system-user-list"
4、toolbar工具栏ID命名规则：id="tb-system-user-list"
5、dialog对话框ID命名规则：id="dlg-system-user-save"
6、form表单ID命名规则：id="fm-system-user-save"
选择一行数据提交(适用于删除单行数据或修改单行数据属性值等操作) lazy.easyui.selected(gridId,url,primarykeyName)
选择多行数据提交(适用对多行数据删除或修改多行数据属性值等操作) lazy.easyui.selections(gridId,url,primarykeyName)
查询并导出结果集,formId用于拼接参数 lazy.easyui.exportSearch(url,formId)
依据一行选中数据的主键，导出结果集 lazy.easyui.exportSelected(gridId,url,primarykeyName)
打开填报数据对话框 lazy.easyui.openDialog(dialogId,_func)
打开修改数据对话框 lazy.easyui.editDialog(gridId,dialogId,formId,_func)
关闭对话框，清除form表单内容 lazy.easyui.closeDialog(dialogId,formId)
提交数据，关闭对话框，刷新数据表格 lazy.easyui.submitDialog(gridId,dialogId,formId,formClearRest)
删除一行数据 lazy.easyui.delete(gridId,url,primarykeyName)
刷新数据表格的内容 lazy.easyui.reload(gridId)
使用url提交待修改的数据或分页查询，并更新结果集 lazy.easyui.urlReload(gridId,url)
使用form提交待修改的数据，并更新结果集 lazy.easyui.formReload(gridId,formId)

	//数据返回成功后，要执行的方法。
});

	//数据返回成功后，要执行的方法。
});
